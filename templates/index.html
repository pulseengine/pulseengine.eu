What is PulseEngine?

A complete WebAssembly Component Model engine built in Rust. Interpret components directly, fuse multiple components into one, optimize them, or transcode them to native ARM for bare-metal embedded targets. One verified pipeline from first byte to final binary — targeting automotive, medical devices, aerospace, and anywhere correctness is not optional.

Every transformation — fusion, optimization, transcoding, execution — is designed to carry mechanized proofs. Rocq, Kani, Z3, Verus. Sigil binds it together with cryptographic attestation: signed, verifiable evidence that your component went through exactly the pipeline you specified.

The Pipeline

A .wasm component enters on the left. Each stage transforms it — fuse, optimize, transcode, fire — and Sigil attests every step. Click any stage to see its source.

{% include "shortcodes/pipeline.html" %}
Meld fuses multiple components into a single module — runtime linking eliminated.
Loom runs twelve optimization passes via Cranelift's ISLE engine.
Synth transcodes Wasm to native ARM through program synthesis — not translation, synthesis.
Kiln interprets and executes components — full Component Model, WASI 0.2, no_std.
Sigil signs and attests every transformation — end-to-end verifiable.
